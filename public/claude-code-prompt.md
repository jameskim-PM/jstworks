> **중요 — 디자인 가이드라인**
> 이 작업은 **기존 홈페이지의 디자인 스타일을 그대로 유지**하면서 새로운 섹션을 추가하는 것입니다.
> - 새로운 색상, 폰트, 컴포넌트 라이브러리를 임의로 도입하지 마세요.
> - 기존 페이지에서 사용 중인 컬러 토큰, 타이포그래피 스케일, 여백 규칙, 카드/버튼 스타일을 **그대로 따라서** 신규 섹션을 구성하세요.
> - 기존 컴포넌트가 있으면 재사용하고, 새 컴포넌트를 만들 때도 기존 명명 규칙과 파일 구조를 따르세요.
> - 회사소개서 용도이므로 **긍정적이고 신뢰감 있는 톤**으로 작성합니다.

## 2. VISION 섹션 추가 (`#vision`)

### 섹션 구조

전체 레이아웃은 **좌측 이미지 / 우측 텍스트** 2단 구성입니다. 모바일에서는 위아래로 스택됩니다.

```
┌─────────────────────────────────────────────────────────┐
│  Vision Statement (Joshua Tree)                         │
│  "혹한의 사막에서도 마르지 않는 생명력으로,                  │
│   K-Heritage를 미국 땅에 깊게 뿌리내리다"                   │
│                                                         │
│  ┌─────────────────┐    ┌──────────────────────────┐    │
│  │                 │    │ 열매 (What?)              │    │
│  │                 │    │   고객이 즐길 최종 결과물? │    │
│  │   [Joshua Tree  │    │   • 합리적 프리미엄 ...    │    │
│  │     이미지      │    │   • 종합 K-Pop Culture..  │    │
│  │     슬롯]       │    │   • 원스탑 쇼핑 경험      │    │
│  │                 │    │                          │    │
│  │   ← 이 자리에   │    │ 줄기 (How?)               │    │
│  │     이미지      │    │   어떻게 현실로 만드는가?  │    │
│  │   삽입 예정     │    │   • 스토리 중심 디지털... │    │
│  │                 │    │   • 트렌드 큐레이션 속도 │    │
│  │                 │    │   • 투트랙 유통 전략     │    │
│  │                 │    │                          │    │
│  │                 │    │ 뿌리 (Why?)               │    │
│  │                 │    │   왜 K-GANADA인가?        │    │
│  │                 │    │   • 한국 Heritage 전파   │    │
│  │                 │    │   • 압도적 진정성        │    │
│  │                 │    │   • 서울의 바이브        │    │
│  └─────────────────┘    └──────────────────────────┘    │
└─────────────────────────────────────────────────────────┘
```

### 이미지 슬롯

좌측 이미지 영역은 **빈 자리(placeholder)** 로 만들어 두세요.
- 권장 비율: 4:5 또는 1:1 (정사각형~세로형)
- `<img src="" alt="Joshua Tree" />` 형태로 두고, src는 비워두거나 `/images/vision-joshua-tree.jpg` 같은 경로만 잡아둡니다.
- 추후 사용자가 직접 조슈아트리 사진을 업로드해서 교체할 예정입니다.
- 이미지 모서리는 기존 페이지의 다른 이미지들과 동일한 radius를 적용합니다.

### 텍스트 콘텐츠 (그대로 사용)

**상단 헤더**
- H2: `Vision Statement (Joshua Tree)`
- 서브헤더 (이탤릭): `"혹한의 사막에서도 마르지 않는 생명력으로, K-Heritage를 미국 땅에 깊게 뿌리내리다"`

**열매 (What?) — 고객이 즐길 최종 결과물은 무엇인가?**
- 합리적 프리미엄 (Affordable Premium) 고퀄리티 제품
- 종합 K-Pop Culture 포트폴리오 (Comprehensive Categories)
- 원스탑 쇼핑 경험 (한국제품 → K-GANADA)

**줄기 (How?) — 어떻게 현실로 만드는가?**
- 스토리 중심의 디지털 내러티브 (Digitally Narrative)
- 트렌드 큐레이션 속도 (Curated Trend-Speed)
- 투트랙 유통 및 커뮤니케이션 전략

**뿌리 (Why?) — 왜 K-GANADA인가?**
- 한국 Heritage의 강인한 전파 (조슈아트리의 생명력)
- 압도적 진정성 (Absolute Authenticity)
- 서울의 바이브 (Seoul Vibe)

### 디자인 디테일

- "열매 / 줄기 / 뿌리" 각 카테고리는 시각적으로 구분되는 블록(카드 또는 구분선)으로 처리합니다.
- 각 카테고리 제목 옆의 영문 `(What?) / (How?) / (Why?)` 는 포인트 컬러로 강조 (기존 페이지의 액센트 컬러 사용).
- 각 항목은 글머리표(`•` 또는 dash) 리스트로 표시.
- 좌측 이미지에서 우측 카테고리로 향하는 **화살표 또는 연결선**은 데스크톱에서만 표시하고, 모바일에서는 생략하여 깔끔하게 정리합니다.

## 3. MARKET 섹션 추가/업데이트 (`#market`)

### 섹션 메시지 (헤드라인)

> **"미국 시장에서 K-Culture를 전하는 두 개의 축"**
> JSTWorks는 미국 소비자를 대상으로 한국의 문화, 상품, 패션, 푸드 등 K-Culture 전 영역을
> **Creative(콘텐츠)** 와 **Place(유통)** 두 개의 트랙으로 전개합니다.

### 레이아웃: 2-Track 카드 구성

좌우 2단 또는 상하 2개 카드로 구성합니다. 각 카드는 헤더 + 채널 리스트로 구성됩니다.

---

### Track 1 — Creative (콘텐츠 & 커뮤니케이션)

**역할:** 창작물을 통해 K-Culture의 매력을 전달하고, 고객을 K-GANADA로 유인합니다.
(스토리 중심의 디지털 내러티브 / 광고 커뮤니케이션 최적화)

**운영 채널**
- **YouTube** — 롱폼 스토리텔링, 브랜드 다큐멘터리, 제품 리뷰
- **Instagram** — 비주얼 캠페인, 릴스, 라이프스타일 큐레이션
- **TikTok** — 숏폼 트렌드 콘텐츠, 챌린지, 바이럴 마케팅
- **Threads** — 실시간 소통, 커뮤니티 빌딩
- **Pinterest** — 비주얼 디스커버리, 무드보드, 한국 라이프스타일 영감

---

### Track 2 — Place (유통 & 판매)

**역할:** 미국 전역의 주요 커머스 플랫폼에 K-Culture 상품을 폭넓게 노출하고,
원스탑 쇼핑 경험을 제공합니다. (세일즈 커뮤니케이션 최적화 / CRO 최적화)

**오픈마켓 / 글로벌 커머스**
- **Amazon** — 미국 최대 커머스 플랫폼, 핵심 매출 채널
- **Etsy** — K-Craft, 굿즈, 디자인 상품 특화
- **TikTok Shop Center** — 콘텐츠-커머스 연계 라이브 셀링
- **Meta Shop (Instagram Shop)** — 소셜 커머스
- **eBay** — 글로벌 리치 확대

**자사몰 (D2C)**
- **K-GANADA (Shopify)** — 자체 브랜드 스토어
- **JSTWorks 회사 홈페이지** — 브랜드 허브

---

### 핵심 전략 메시지 (섹션 하단)

> **"하나의 일관된 메시지로, Creative와 Place를 통합 운영합니다."**
> — IMC(Integrated Marketing Communications) 전략 기반
> — Creative에서 광고 커뮤니케이션, Place에서 세일즈 커뮤니케이션 최적화
> — 미국 소비자에게 K-Culture의 진정성과 트렌드를 함께 전달

### 디자인 디테일

- 각 채널은 작은 카드/태그 또는 로고 아이콘 + 채널명 + 한 줄 설명 형태로.
- Creative 카드와 Place 카드의 액센트 컬러를 살짝 다르게 (기존 페이지 컬러 팔레트 내에서) 구분.
- 모바일에서는 세로 스택, 데스크톱에서는 2단 그리드.
- 자사몰(K-GANADA Shopify + 회사 홈페이지)은 Place 카드 안에서 별도 구분선으로 묶어 강조.
